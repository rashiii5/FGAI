# Lab 1: Exploring Generative AI Tools for Text, Image & Video Creation

## Objective
The objective of this lab is to understand and apply the fundamentals of prompt engineering by using generative AI tools to create academic promotional content. This includes designing effective prompts for text generation, AI-based image/poster generation, and AI video creation for a university-level workshop/camp announcement. The lab aims to learn how role definition, task clarity, context, and constraints help in obtaining accurate and professional AI outputs.

## Tools Used
- ChatGPT (Text generation)
- ChatGPT (Poster generation)
- invideo (Video generation)

## Tasks Completed
1. Generated formal launch email for Blood Donation Camp
2. Created AI digital poster for academic promotion
3. Created short AI promotional video with voiceover
